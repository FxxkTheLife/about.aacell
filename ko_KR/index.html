웹 기반 교차 플랫폼 파일 공유 웹 사이트:
작은 방에서 자유롭게 파일을 공유하세요!

파일 전송은 단일 파일 만 가능합니까?

작은 방이 있으니 여러 개의 문서를 쉽게 저장할 수 있다.

파일 크기 제한?

현재 제한은 없지만 단시간에 너무 많은 데이터를 서버에 공급하지 마십시오. 지원하지 못할 수 있습니다.

계정에 로그인하는 데 너무 많은 문제가 있습니까?

방 ID는 짧고 기억하기 쉬우 며 임의의 숫자를 이름으로 사용할 수 있으며 관리자 암호는 파일 관리에 사용되며 방 키는 방에 들어가는 데 사용됩니다.

가끔 문자 만 보내야하나요?

작은 방에는 채팅 기능이 포함되어있어 문자를 보낼 수 있습니다.

작은 방을 사용할 수있는 제한된 시간은 무엇입니까?

작은 방은 배터리 1 개의 전원을 가지고 있으며 기본 전원은 2 시간 동안 사용할 수 있으며 배터리가 소진되면 시스템에 의해 작은 방 전체가 자동으로 파괴됩니다.

성명서 :이 웹 사이트의 서비스를 이용하려면 "이용 약관"에 동의하고 국가 규정에 위배되는 정보의 유포를 거부하며 기본 사회 윤리 및 인터넷 윤리를 준수해야합니다.
