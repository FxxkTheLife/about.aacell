基於 Web 的跨平台文件共享網站：
擁有小室一間，自在共享文件！

文件傳輸只能單文件？

有了速遞站，輕鬆暫存多個文件。

文件大小限制？

目前未設限，但請不要短時間餵給服務器太多數據，它可能撐不住。

登錄賬戶太麻煩？

速遞站 ID 簡短易記，還可使用隨機號碼作為名稱；管理者密碼用於管理文件，進入密碼用於訪客進入。

有些時候只想傳文本？

速遞站自帶聊天功能，可發送文本。

速遞站的使用時間的限制是什麼？

速遞站貨車具有一節電池的電量，電量默認使用時間是 7 天，電池電量耗盡時，整個速遞站被系統自動摧毀。

聲明：使用本網站服務，即同意《用戶協議書》、拒絕傳播任何違反國家規定的資料、遵守基本社會公德和互聯網道德規範。

鄭重承諾：本站內容均存儲於雲服務器，每個小室中的資源只為進入小室的用戶所使用，其他人無權查看，一個小室的資源在這個小室銷毀時被全部刪除。

小提示：我們致力於為用戶提供穩定、高效、便捷的文件傳輸空間，目前網站仍繼續在開發與完善，若未能及時提供服務敬請諒解！有遇到問題或有更多的建議的朋友，可以加入 QQ 群或者直接聯繫我們，歡迎反饋~
